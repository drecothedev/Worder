This is a very simple app that i decided to program in order to work on traditional data structure algorithims and iOS development. Instead of trying to create this massive new idea, i wanted to focus on something very specific and practical. i chose to create an algorithim that implements a Trie to show word suggestions when the user is typing. Think of auto-correct or text predictions. The main focus of this project is to implement a Trie within a user experience.

## Time Complexity
All methods within Trie will have O(n) and O(1) where n represents the length of the word.
